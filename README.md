DESCRIPCION DEL PROYECTO

Este proyecto fue desarrollado en Java utilizando JSF, con Bootstrap, CSS externo, con el objetivo de aplicar lo visto en clase sobre desarrollo web con Java, manejo de formularios, validaciones y uso de los demas temas.

El sistema está dividido en dos ejercicios principales:

Simulador CDT
Registro de Aspirantes Universitarios

Ambos ejercicios nos permiten ingresar datos, validarlos y mostrar resultados en pantalla usando JSF.
